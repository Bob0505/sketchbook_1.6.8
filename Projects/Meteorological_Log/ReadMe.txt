Environment:
* Arduino V1.6.x / V1.8.x
* Check library version
	Sketch -> Include Library -> Manage Libraries
	Search keyword "" ....
		"Wire"	Built-in by Arduino V1.0.0
		"SPI"	Built-in by Arduino V1.0.0
		"SD"	Built-in by Arduino, SparkFun V1.1.1 
		"Rtc by Makuna" v1.0.1				& install is this
		"Adafruit Htu21d Library" v1.0.0	& install is this
* Open File
	File -> Open...
		[Download dir]\Meteorological_Log\Meteorological_Log.ino
* Set Up Tools
	Tools -> Board: "xxxx" -> Arduino Nano
	Tools -> Processor: "xxxx" -> ATMega328
	Tools -> Port: "comx" -> choose your debug/download com port
* Compile & Upload
	Sketch -> Upload or [Ctrl + U]
